# Contributing

Thanks for your interest in contributing.

## Support Disclaimer
This project is provided **as-is**. Support is not guaranteed. Issues and pull requests may not receive responses or fixes.

## How to Contribute
- Open an issue describing the change or bug.
- If you submit a PR, keep it small and focused.
- Include clear reproduction steps for bugs.

## Development
```bash
npm install
npm run dev
```

## Code Style
- Run `npm run lint` before submitting.
